# Keybindings


## Hammerspoon - [ Meta + Space ]

    * Ace switch window           [ Leader + W ]
    * Fit window to grid          [ Leader + G ] 
    * Fullscreen window           [ Leader + HL ]
    * Center window               [ Leader + JK ]
    * Move to top right           [ Leader + LK ]
    * Move to bottom right        [ Leader + LJ ]
    * Move to top left            [ Leader + HK ]
    * Move to bottom left         [ Leader + HJ ]
    * Shrink window left          [ Leader + Shift-Cmd-H ]
    * Shrink window bottom        [ Leader + Shift-Cmd-J ]
    * Shrink window top           [ Leader + Shift-Cmd-K ]
    * Shrink window right         [ Leader + Shift-Cmd-L ]
    * Grow window left            [ Leader + Cmd-H ]
    * Grow window bottom          [ Leader + Cmd-J ]
    * Grow window top             [ Leader + Cmd-K ]
    * Grow window right           [ Leader + Cmd-L ]
    * Enlarge window              [ Leader + = ]
    * Shrink window               [ Leader + - ]
    * Move window left            [ Leader + Shift-H ]
    * Move window down            [ Leader + Shift-J ]
    * Move window up              [ Leader + Shift-K ]
    * Move window right           [ Leader + Shift-L ]
    * Move to next screen         [ Leader + ; ]
    * Move to left 1/3rd          [ Leader + Ctrl-H ]
    * Move to left 2/3rds         [ Leader + Ctrl-J ]
    * Move to right 1/3rd         [ Leader + Ctrl-L ]
    * Move to right 2/3rds        [ Leader + Ctrl-K ]
    * Cycle app windows           [ Cmd-Shift-~ ]

    * Invert colors               [ Leader + I ]
    * Toggle redshift             [ Leader + R ]
    * Activate screensaver        [ Leader + S ] 

## tmux - [ Ctrl + Space ]

    * Create new window           [ Leader + C ]
    * Rename current window       [ Leader + , ]
    * Kill current window         [ Leader + & ]
    * Next window                 [ Leader + N ]
    * List windows                [ Leader + W ]
    * Kill focused pane           [ Leader + X ]
    * Split window vertically     [ Leader + V ]
    * Split window horizontally   [ Leader + S ]
    * Toggle maximize current     [ Leader + Z ]
    * Switch to pane left         [ Meta-H ]
    * Switch to pane down         [ Meta-J ]
    * Switch to pane up           [ Meta-K ]
    * Switch to pane right        [ Meta-L ]
    * Switch to window #          [ Leader + {0-9} ]
    * Enter normal mode           [ Leader + Space ]
    * Enter copy mode             [ V ]
    * Exit copy mode              [ Q ]
    * Paste from copy mode        [ P ]
    * Scroll up in copy mode      [ Shift-K ]
    * Scroll down in copy mode    [ Shift-J ]
    * Clear terminal              [ Cmd-K ]
    * Scroll clear terminal       [ Ctrl-L ]
